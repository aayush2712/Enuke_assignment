# Enuke Assignment

> Backend API for authenticated users to upload the image of maximum size of 500 KB.

## Usage

Rename "config.env.env" to "config.env" and update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
